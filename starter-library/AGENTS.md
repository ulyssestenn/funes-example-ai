# Starter Library — Librarian-managed knowledge base

**Scope:** foundational large-language-model research papers (2017–2025) —
the Transformer and its descendants, scaling and emergence, alignment and
preference learning, and LLM reasoning.

**Operating manual:** follow the shared **[Librarian Protocol](../protocol.md)**
(ingest → compile → Q&A → health-check, conventions, templates). Link style:
markdown relative links. No deviations from the standard structure.

Human usage and layout: see [`README.md`](./README.md).
